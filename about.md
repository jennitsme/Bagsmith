## Project Name
Bagsmith

## Short Description
Bagsmith is an AI-powered builder pipeline on Bags that turns prompts into real on-chain execution flows. Users can fetch live quotes, execute swaps, and track results in a real analytics dashboard with exportable traction evidence.

## What did you build?
We built a full-stack Next.js app integrated with Bags API:
- Quote retrieval (`/trade/quote`)
- Swap transaction creation (`/trade/swap`)
- Signed transaction submission (`/solana/send-transaction`)
- Persistent run logging (success/failure/signature)
- Analytics dashboard with time filters and CSV export

## Why is it useful?
Most hackathon demos stop at mock UIs. Bagsmith provides a real, auditable path from idea to on-chain execution and measurable usage proof. It helps teams and creators validate mini-app mechanics quickly with transaction-level evidence.

## Bags integrations used
- Bags Public API v2
- Trade quote endpoint
- Swap creation endpoint
- Solana send transaction endpoint

## Traction / Evidence
- Real on-chain transaction signatures generated by the app
- Run logs with timestamps, modes, and outcomes
- Analytics metrics (total runs, execute runs, success rate)
- CSV export for external verification

## What’s next?
- Non-custodial signing flow / wallet delegation
- User auth and per-user workspace isolation
- Richer agent orchestration for app template generation
- Production-grade security hardening and monitoring
